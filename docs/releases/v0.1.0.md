# 0.1.0

`0.1.0` is the first public-ready release of graphobs.

## Highlights

- Runtime-independent contracts for public and private graph state boundaries.
- LangGraph node and subgraph wrappers that project input, validate output, and
  emit spans.
- OpenTelemetry tracing helpers with OpenInference-compatible attributes.
- Compact-by-default payload serialization with explicit full mode.
- Structured lifecycle logging helpers with correlation fields.
- Synthetic examples for retrieval, subgraph boundaries, tool-like flows, and
  backend export shape.
- Adoption docs for one-node migration, payload safety, and backend
  portability.

## Compatibility

- Python 3.11 through 3.14.
- LangGraph 1.2.5 or newer.
- OpenTelemetry API 1.42.1 or newer.

## Release Boundary

This release prepares local package artifacts and release notes. Creating the
remote repository, tag, package publication, and hosted release remains a
maintainer action.
