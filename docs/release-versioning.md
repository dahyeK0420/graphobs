# Release And Versioning Notes

Graph Observability Kit will use Semantic Versioning once releases begin.

## Initial Version

`0.1.0` is the first public-ready release.

## Planned Release Flow

- Patch releases fix defects without changing public contracts.
- Minor releases add compatible functionality.
- Major releases may change public contracts and require migration notes.

## Changelog

Human-readable changes are tracked in `CHANGELOG.md`.

## Release Notes

Release summary pages live under `docs/releases/`. GitHub release-note copy is
prepared locally under `.github/release-notes/`, but creating tags, hosted
releases, and package publications remains a maintainer action.
