# Package

The package root exposes the headline adoption path: `NodeContract`,
`contract_node`, `add_contract_node`, `build_invoke_config`, and `__version__`.
Use the focused submodule reference pages for contract primitives, LangGraph
helpers, structured logging classes, and tracing helpers.
`contract_node` can wrap a function directly or decorate it with
`@contract_node(contract)`.
