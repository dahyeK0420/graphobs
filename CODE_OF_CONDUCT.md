# Code Of Conduct

## Our Pledge

We want this project to be a respectful, practical, and welcoming place for contributors with different backgrounds and experience levels.

## Expected Behavior

- Be considerate and direct.
- Assume good intent while still addressing impact.
- Keep technical criticism focused on code, docs, design, and outcomes.
- Respect boundaries, privacy, and time.
- Help make discussions useful for future readers.

## Unacceptable Behavior

- Harassment, threats, insults, or sustained disruption.
- Discriminatory language or conduct.
- Publishing private information without clear permission.
- Unwanted sexual attention.
- Any conduct that would make reasonable participation unsafe or unwelcoming.

## Enforcement

Project maintainers may remove comments, close issues, decline contributions, or restrict participation when needed to protect the project community. Reports should be sent through the security or maintainer contact path documented for the public repository.
